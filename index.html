Projects & Experience

멀티 클라우드 웹 애플리케이션 배포

AWS EC2와 Azure VM을 활용하여 동일한 웹 애플리케이션을 Docker 컨테이너로 배포하는 프로젝트를 진행했습니다. Load Balancer를 통한 트래픽 분산과 RDS, Azure SQL을 활용한 데이터베이스 이중화를 구현했습니다.

Kubernetes 클러스터 구축 및 관리

로컬 환경에서 minikube를 활용한 Kubernetes 클러스터를 구축하고, 다양한 애플리케이션을 Pod로 배포하여 오케스트레이션을 학습했습니다. Service, Ingress, ConfigMap 등의 리소스를 활용한 실습을 진행했습니다.

클라우드 비용 최적화 스터디

AWS와 Azure의 비용 구조를 분석하고, Reserved Instance, Spot Instance 등을 활용한 비용 최적화 방안을 연구했습니다. 실제 사용량 기반의 비용 시뮬레이션을 진행했습니다.

Learning Journey

클라우드 입문 (6개월 전)

AWS 기초 개념 학습을 시작으로 클라우드 컴퓨팅의 핵심 원리를 이해했습니다. EC2, S3 등 기본 서비스들을 직접 사용해보며 클라우드의 매력을 느꼈습니다.

컨테이너 기술 학습 (4개월 전)

Docker를 통해 컨테이너 기술을 익히고, 애플리케이션 배포의 효율성을 경험했습니다. 이후 Kubernetes를 학습하며 대규모 컨테이너 관리의 필요성을 이해했습니다.

멀티 클라우드 확장 (2개월 전)

AWS에서 Azure로 학습 범위를 확장하며 멀티 클라우드 환경의 장점을 학습했습니다. 각 플랫폼의 특성을 비교 분석하며 최적의 솔루션을 선택하는 능력을 기르고 있습니다.

인프라 자동화 도전 (현재)

Terraform과 Ansible을 활용한 Infrastructure as Code를 학습하고 있습니다. 반복적인 인프라 구축 작업을 자동화하여 효율성을 높이는 방법을 연구하고 있습니다.
